# Objetivo: Crear una Aplicación Web de cave crawler.

## 1. Estructura de Archivos:

- .antigravity/prompts.txt: listado de prompts relacionados al proyecto.
- .antigravity/rules.md: reglas de desarrollo.
- .antigravity/specs.md: lista de especificaciones del proyecto.

- src/index.html: Contenedor base.
- src/script.js: Apartafo funcional.
- src/style.css: Apartado estético.

## 2. Requisitos de Código (Reglas de Oro):

- Modularidad: El código en js se distribuye en funciones puras.
- Punto de entrada: Una función main() en script.js que orqueste la producción.
