# Product Context

## Visión del Proyecto
Firewall Hero es un programa de tests (trivia) sobre sistemas y redes, diseñado como una experiencia gamificada estilo "autoescuela" para aprender conceptos de hardware, sistemas operativos, comandos de Linux/Windows, redes IP y contenedores.

## Objetivos
- Gamificar el aprendizaje de conceptos de sistemas informáticos.
- Proveer una interfaz atractiva y responsiva (simulando una topología de red).
- Asegurar que el conocimiento del usuario se ponga a prueba bajo presión (con un límite de tiempo).

## Casos de Uso
1. El usuario navega a través de nodos de red.
2. Cada nodo puede contener una trivia, un objeto de recuperación o un refugio seguro (históricamente se probaron otros minijuegos como Simon, pero se descartaron por problemas de flujo).
3. El usuario debe responder correctamente a las preguntas de sistemas para poder continuar sin perder integridad (vidas).
